CSS Notes

The purpose of CSS is

  • CSS can be included within html elements using the style attr
  • CSS can be used with the style html element
  • CSS is also used for altering colors, fonts, backgrounds, etc.
  • CSS can implement some effects and animations

CSS Notes

Purpose of CSS

  • To give the outside of the site a unique look and feel.
  • Alter color, fonts, backgrounds, etc.
  • To implement some effects and light animations.

CSS Statements

Four main components include the following:

  1. Declaration: the statement as a wonle with curly braces.
  2. Selector: the name of the part you want to style.
  3. Property: properties to edit(e.g., width, height, background, etc.)
  4. Value: numbers, colors, urls - values that we assign.
